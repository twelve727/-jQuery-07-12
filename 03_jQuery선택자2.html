prop() 메소드

        attribute란? 
        type, name, class, id, value 등과 같이
        속성 값을 별도로 입력해야 하는 속성

        property : checked, selected 와 같이
        속성값이 별도로 입력되지 않는 속성

        prop("속성명") : 해당 속성이 요소에 존재하면 true,
        아니면 false를 반환

        prop("속성명", true | false) : 해당 속성을 checked,
        selected 상태로 변경(true) / 해제(false)
취미 : 풋볼축구 게임 음악감상
